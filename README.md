# CALCULATOR

## Problem Description

A program that can make simple mathematical operation, like:
- add
- subtract
- clear

## Dev environment setup

- [Java JDK 13](https://www.oracle.com/java/technologies/javase-jdk13-downloads.html)
- [Gradle](https://gradle.org/install/)

## Build instructions

```
./gradlew clean build
```

## Run instructions

```
./gradlew run
```

## Run test
```
./gradlew clean test
```
